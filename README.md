## Rifas pro Yapira

### Pré-requisitos:
#### Instale os modulos 'tkinter' e 'cv2':

```bash
pip install tk
pip install opencv-python
```

### Uso:
#### Só rodar com 
```bash
python3 faz_rifa.py
```

### !! Só funciona nesta resolução !! 
#### Se você deixar o script junto com a `rifa.png` local, ele a detecta automaticamente.
